#  Inventory Management System

A full-stack **Inventory Management System** built to efficiently manage products, categories, orders, and users with secure authentication and real-time inventory tracking. Designed with scalability, security, and usability in mind.

##  Features

-  Admin Login & Authentication  
-  Role-Based Access Control (Admin / User)  
-  Product Management  
-  Category Management  
-  Order Management  
-  User Management  
-  Full CRUD Functionality (Create, Read, Update, Delete)  
-  Real-Time Inventory Tracking  
-  Responsive UI (Desktop & Mobile)  
-  Secure Server-Side Processing  
-  Secure Database Interaction

##  Deployment

The application is deployed on a **DigitalOcean Ubuntu server** using **Apache** as the web server.

- OS: Ubuntu  
- Web Server: Apache  
- Environment: Production-ready  
- Security: Protected routes, server-side validation  

---

##  User Roles

### Admin
- Full access to the system
- Manage products, categories, orders, and users
- Monitor and control inventory

### User
- Limited access based on assigned permissions
- View and interact with allowed data only
